# 补问问题清单（Q1..Q6）

> 使用说明：当信息不足时，从下列问题里选“最少必要项”一次性提问。

- Q1 页面受众是谁？核心决策问题是什么？
- Q2 原型里有哪些模块（图、表、卡片、筛选）？按什么顺序展示？
- Q3 每个模块分别依赖哪个接口？接口 method/url/params 是什么？
- Q4 接口返回字段中，哪些字段映射到哪些 UI 元素？
- Q5 交互要求是什么（筛选、排序、分页、时间范围、刷新）？
- Q6 技术边界与输出协议是什么（原生HTML/React、单文件/多文件、是否只返回代码）？

## 提问规则

- 只提可直接回答的问题。
- 不问开放式主观题。
- 优先消除实现阻塞项（接口与映射 > 样式偏好）。
